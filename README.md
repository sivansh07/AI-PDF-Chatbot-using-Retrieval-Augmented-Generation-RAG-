# AI-PDF-Chatbot-using-Retrieval-Augmented-Generation-RAG-
AI PDF Chatbot using Retrieval-Augmented Generation (RAG) built with Python, LangChain, FAISS, Hugging Face Embeddings, Google Gemini, and Streamlit. Upload PDFs, perform semantic search, and get accurate, context-aware answers through an interactive conversational interface powered by vector similarity search.
